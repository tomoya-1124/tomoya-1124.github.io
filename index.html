About

東京拠点のソフトウェアエンジニア。業務では Java / Spring Boot を中心に、フロントは Knockout.js、DB は Oracle / PostgreSQL を扱っています。既存機能の改修・テスト設計・API実装を通して「読みやすく安全なコード」と「再現性のある検証」を意識しています。

大学で経済を学んだ後、論理的に物事を組み立てることが得意だと気づき、問題解決を形にできるソフトウェア開発に惹かれました。

  • 得意: 仕様の分解、境界値テスト設計、ログ調査
  • 興味: CI/CD、パフォーマンス、設計の標準化
  • 個性: 音楽とファッションが好き。実務の学びをブログで継続発信。

今後は フルスタックエンジニア としての経験を積み、設計から運用までを一貫して担える力を伸ばしていきたいと考えています。特に CI/CDやクラウド技術 を取り入れ、よりスケーラブルで再現性の高いシステム構築に挑戦することが目標です。
